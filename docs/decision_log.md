# Decisions

## 2026-07-10

### Decision

Adopt a research-oriented repository structure.

### Motivation

The project requires not only software development but also scientific documentation, experimental protocols, and reproducible research workflows.

### Alternatives Considered

A traditional code-only repository.

### Expected Impact

Improved project organization, reproducibility, and future publication workflow.

---

## 2026-07-11

### Decision

Develop a multimodal architecture combining IMU, computer vision, OpenSim, and artificial intelligence.

### Motivation

Different sensing modalities provide complementary information.

IMU systems provide accurate temporal motion information, while computer vision enables markerless and scalable analysis.

### Alternatives Considered

Using only IMU-based analysis.

Using only camera-based pose estimation.

### Expected Impact

Improved robustness and more comprehensive biomechanical interpretation.
