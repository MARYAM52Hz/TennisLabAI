# References

## Overview

This document serves as the central bibliography for the TennisLabAI project.

It collects scientific publications, datasets, software libraries, documentation, standards, and online resources referenced throughout the project.

The objective is to maintain a consistent and reproducible reference system across documentation, experiments, and future publications.

---

# Citation Style

The preferred citation style for this project is:

* IEEE (recommended for engineering publications)

Alternative styles may be adopted when required by specific journals or conferences.
