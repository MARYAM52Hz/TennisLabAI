# IMU Processing

## Overview

This module processes wearable inertial sensor data acquired from the Noraxon Ultium myoMOTION system.

## Objectives

- Import IMU recordings
- Synchronize sensor streams
- Estimate joint kinematics
- Extract biomechanical variables

## Expected Inputs

- Raw IMU recordings
- Calibration files
- Subject metadata

## Outputs

- Joint angles
- Angular velocity
- Linear acceleration
- Segment orientation
- Range of motion (ROM)

## Future Work

- Sensor calibration
- Drift correction
- Signal filtering
- Feature extraction
