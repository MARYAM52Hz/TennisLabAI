# Pose Estimation

## Overview

This module estimates the human body pose from RGB videos using markerless computer vision.

## Objectives

- Detect 2D body landmarks
- Reconstruct 3D human pose
- Track body joints during tennis movements
- Generate biomechanical features for further analysis

## Planned Methods

- MediaPipe Pose
- RTMPose
- OpenPose
- Pose2Sim
- OpenCap (optional)

## Inputs

- RGB videos
- Multi-camera recordings

## Outputs

- 2D keypoints
- 3D joint coordinates
- Skeleton visualization
- Pose confidence scores

## Future Work

- Multi-view pose fusion
- Occlusion handling
- Temporal pose smoothing
