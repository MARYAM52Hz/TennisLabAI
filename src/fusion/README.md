# Sensor Fusion

## Overview

This module combines wearable IMU data with markerless computer vision to improve biomechanical estimation.

## Objectives

- Synchronize multiple sensors
- Align coordinate systems
- Fuse complementary information
- Improve robustness and accuracy

## Inputs

- IMU features
- Camera-based pose estimation
- Time synchronization signals

## Outputs

- Fused joint kinematics
- Corrected trajectories
- Confidence estimates

## Future Work

- Kalman filtering
- Deep sensor fusion
- Real-time synchronization
